# Playwright Project

## Description
This project uses [Playwright](https://playwright.dev/) to automate the Chromium browser. Test scripts are automatically generated by AI after providing it with a prompt describing the test steps.

## Folder Structure
- `descriptions/` – Description files
- `testcases/` – Generated test cases
- `playwright.config.js` – Playwright configuration

## Installation Guide

### 1. Install Node.js
- Download and install Node.js from [nodejs.org](https://nodejs.org/)

### 2. CLI Commands
Run the following command to allow script execution:
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### 3. Install Playwright and Chromium
Run the following commands in the project directory:
```powershell
npm install --save-dev playwright
npm install --save-dev @playwright/test
npx playwright install chromium
npx playwright install winldd
```

### 4. Run Playwright
```powershell
npx playwright test
npx playwright test --headed
npx playwright test --grep <tag>
```
